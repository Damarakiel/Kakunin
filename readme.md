#Kakunin - automated testing framework by TSH & Takamol

##Requirements:

1. `node v7.0+`
2. `JDK` installed
3. `Chrome` installed

##How to test?
1. Run `npm run test` to execute both units and functional tests.

##How to build?

1. Run `npm run build` to build
